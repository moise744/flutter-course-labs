# Lab 2 – Flutter Basics

## Overview

This lab introduces the fundamentals of Flutter development. The objective is to understand the structure of a Flutter project and build a simple user interface using basic widgets.

## Description

In this lab, a basic Flutter application was created that demonstrates:

- Use of Scaffold as the main layout structure
- An AppBar at the top of the screen
- Text widgets to display content
- Simple layout organization using Column and Center
- Basic UI styling

This lab provides a foundation for more advanced Flutter concepts in future labs.

## How to Run

1. Navigate to the project directory:
   cd hello_world

2. Get dependencies:
   flutter pub get

3. Run the application:
   flutter run -d chrome
